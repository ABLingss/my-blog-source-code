2:22。

在 GitHub 上搜了几个关键词：`QQ-History-Backup`，`Export-Chat-Logs`。网页上的 Star 数并不高，说明这只是极少数人才会有的需求。熟练地 `git clone`，配置环境，然后在终端里输入命令。

屏幕上开始跳动绿色的进度条，一行行解析出的数据在飞速滚动。

手机屏幕亮了。微信群里有人发了截图，是 ACL 的录用通知。几个认识的朋友中稿了，大家在群里刷着"牛逼"和红包。盯着那个绿色的录用邮件截图看了几秒钟，然后机械地回复了一个点赞的表情包。

这种感觉很荒谬。

还站在东疆港的海边，听她食熊则肥，食蛙则瘦的盛夏，讲那个无疾而终的故事，其实不过是几小时前的事情。而现在，我坐在这儿，一边试图用技术手段去打捞自己几年前的碎语，一边被同龄人的学术成就精准地刺痛。

还没来得及消化海风带来的寒意，就被现实的进度条推到了死角。

明天下午四点是组会。

看了看还没怎么动过的 PPT 模板，又看了看正在解析中的聊天记录。终端窗口里跳出了一个报错：`Error: Database is encrypted`。

没有去修那个 Bug，而是把电脑合上了。

9平米的、即使如此也即将失去的宿舍里陷入了彻底的黑暗，舍友早已睡去。并不是因为某种巨大的悲剧，而是一种由于睡眠不足、自我怀疑和对未来的恐慌交织在一起的、具体的窒息感。

生活没打算给我一个缓冲期。

躺在床上，闭上眼，脑子里不是海，也不是那个女孩。
是 ACL 的录用邮件，是组会要用的图表，是那个没跑通的 Python 脚本。

睡不着，但这和浪漫一点关系都没有。
